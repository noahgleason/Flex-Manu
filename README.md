# Flex Manufacturing Website

Static site for Flex Manufacturing, a custom gear and precision machining company in Clinton Township, Michigan (est. 1974). Deployed via GitHub Pages — plain HTML/CSS with minimal vanilla JS, no build step, no framework.

## Pages

- `index.html` — Home
- `capabilities.html` — Capabilities
- `reverse-engineering.html` — Reverse Engineering
- `about.html` — About
- `quote.html` — Request a Quote (form posts to a third-party form service; no backend)

## Structure

- `css/style.css` — shared styles
- `js/main.js` — shared vanilla JS (nav highlighting, etc.)
